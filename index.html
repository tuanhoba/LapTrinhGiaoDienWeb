Môn Lập Trình Giao Diện Web

Giảng viên: Thạc sĩ Hoàng Quốc Tuấn

Mục tiêu khóa học

Môn học Lập Trình Giao Diện Web trang bị cho sinh viên các kiến thức cơ bản và nâng cao về phát triển giao diện web, sử dụng các công nghệ hiện đại trong việc xây dựng các website tương tác và đáp ứng nhu cầu của người dùng. Sau khóa học, sinh viên có thể:

  • Hiểu được cách xây dựng giao diện web chuyên nghiệp, đẹp mắt và dễ sử dụng.
  • Làm quen với các công cụ và thư viện hỗ trợ phát triển giao diện web (HTML, CSS, JavaScript, Bootstrap, jQuery, v.v).
  • Tạo ra các website với khả năng tương tác tốt, tương thích với nhiều thiết bị khác nhau (Responsive Design).
  • Phát triển kỹ năng làm việc nhóm, xây dựng và tối ưu hóa giao diện web.

Điều kiện học

  • Sinh viên cần có kiến thức cơ bản về tin học và sử dụng máy tính.
  • Không yêu cầu kinh nghiệm lập trình trước khi học.
  • Sinh viên cần tham gia đầy đủ các bài học lý thuyết và thực hành.
  • Sinh viên nên có máy tính cá nhân hoặc thiết bị có thể cài đặt phần mềm lập trình và duyệt web.

Nội dung chi tiết khóa học

Bài 1: Giới thiệu về Lập Trình Giao Diện Web

Tổng quan về lập trình web. Các thành phần cơ bản của một trang web: HTML, CSS, JavaScript. Quá trình phát triển giao diện web. Các công cụ hỗ trợ phát triển giao diện web.

Bài lab 1: Làm quen với HTML và cấu trúc của một trang web

  • Xây dựng một trang web đơn giản với HTML.
  • Cấu trúc cơ bản của HTML, thẻ meta, link và script.

Hướng dẫn chi tiết:

  1. Tạo một trang web giới thiệu bản thân với các thông tin cơ bản (tên, tuổi, sở thích, ảnh cá nhân).
  2. Sử dụng các thẻ HTML như <h1>, <p>, <img>, <a>.
  3. Lưu file HTML với tên Lab1_[HọTênSinhViên].html vào ổ D.

Hướng dẫn nộp bài:

  • Đưa file lên GitHub (tạo repo riêng cho môn học).
  • Nộp file HTML gốc lên hệ thống Hệ thống nộp bài.
  • Quay video trình bày bài tập (sử dụng OBS, có camera hình cá nhân) và đăng lên YouTube hoặc Zalo.

Bài 2: HTML - Ngôn ngữ đánh dấu siêu văn bản

  • Các thẻ cơ bản trong HTML: <div>, <span>, <h1>, <p>, <a>, v.v.
  • Các thẻ định dạng văn bản và liên kết.
  • Tạo bảng, form và danh sách trong HTML.

Bài lab 2: Xây dựng trang web với các thẻ HTML cơ bản

Hướng dẫn chi tiết:

  1. Tạo một trang web thông tin lớp học, bao gồm danh sách các thành viên và thời khóa biểu.
  2. Sử dụng các thẻ <table>, <ul>, <ol>, <form> để trình bày nội dung.
  3. Lưu file HTML với tên Lab2_[HọTênSinhViên].html vào ổ D.

Hướng dẫn nộp bài:

  • Đưa file lên GitHub.
  • Nộp file gốc lên hệ thống Hệ thống nộp bài.
  • Quay video trình bày bài tập (OBS, có camera hình cá nhân) và đăng lên YouTube hoặc Zalo.

Bài 3: CSS - Cách tạo kiểu cho trang web

  • Khái niệm về CSS và cách áp dụng vào HTML.
  • Cách sử dụng các thuộc tính CSS: font, màu sắc, căn chỉnh, padding, margin.
  • Các phương pháp lựa chọn phần tử CSS: ID, class, thẻ.

Bài lab 3: Thiết kế giao diện với CSS

Hướng dẫn chi tiết:

  1. Thiết kế một trang web giới thiệu công ty với giao diện đẹp (bao gồm logo, menu, nội dung giới thiệu).
  2. Áp dụng CSS cho các phần tử: thay đổi màu sắc, font chữ, căn chỉnh bố cục.
  3. Lưu file HTML và CSS với tên Lab3_[HọTênSinhViên].html và Lab3_[HọTênSinhViên].css vào ổ D.

Hướng dẫn nộp bài:

  • Đưa cả file HTML và CSS lên GitHub.
  • Nộp file gốc lên hệ thống Hệ thống nộp bài.
  • Quay video trình bày bài tập (OBS, có camera hình cá nhân) và đăng lên YouTube hoặc Zalo.

Bài 4: Responsive Web Design - Thiết kế giao diện đáp ứng

  • Khái niệm về thiết kế web đáp ứng.
  • Media Queries trong CSS.
  • Tạo giao diện thích ứng với các loại màn hình khác nhau (máy tính, điện thoại, máy tính bảng).

Bài lab 4: Thiết kế giao diện web đáp ứng

Hướng dẫn chi tiết:

  1. Thiết kế trang web bán hàng trực tuyến (bao gồm danh sách sản phẩm, giỏ hàng).
  2. Sử dụng Media Queries để giao diện hiển thị tốt trên cả máy tính và điện thoại di động.
  3. Lưu file HTML và CSS với tên Lab4_[HọTênSinhViên].html và Lab4_[HọTênSinhViên].css.

Bài 5: JavaScript cơ bản

  • Giới thiệu về JavaScript và cách sử dụng nó để tạo các tương tác động trên trang web.
  • Các khái niệm cơ bản: Biến, hàm, sự kiện, vòng lặp.
  • Cách kết hợp JavaScript với HTML và CSS.

Bài lab 5: Tạo sự kiện và tương tác trên trang web

Hướng dẫn chi tiết:

  1. Thiết kế một trang web có chức năng tương tác: khi nhấn nút, hiển thị hoặc ẩn một phần tử.
  2. Sử dụng JavaScript để xử lý sự kiện và thay đổi nội dung trên trang.
  3. Lưu file HTML và JavaScript với tên Lab5_[HọTênSinhViên].html và Lab5_[HọTênSinhViên].js.

Bài 6: JavaScript nâng cao

  • Phương thức và đối tượng trong JavaScript.
  • Quản lý sự kiện và thao tác DOM (Document Object Model).
  • Ứng dụng JavaScript trong việc xây dựng các tính năng tương tác phức tạp hơn trên website.

Bài lab 6: Thao tác với DOM và các sự kiện JavaScript

Hướng dẫn chi tiết:

  1. Thiết kế một trang web với form đăng ký, sử dụng JavaScript để kiểm tra và xử lý các dữ liệu nhập vào.
  2. Sử dụng sự kiện để hiển thị thông báo khi người dùng nhập sai thông tin.
  3. Lưu file HTML và JavaScript với tên Lab6_[HọTênSinhViên].html và Lab6_[HọTênSinhViên].js.

Bài 7: Bootstrap - Framework thiết kế giao diện

  • Giới thiệu về Bootstrap và cách sử dụng nó để tạo giao diện web nhanh chóng.
  • Thực hành với các lớp và thành phần Bootstrap: grid system, navbars, buttons, forms.
  • Sử dụng các tiện ích Bootstrap để tạo giao diện đẹp và đáp ứng.

Bài lab 7: Thiết kế giao diện với Bootstrap

Hướng dẫn chi tiết:

  1. Thiết kế một trang web bán hàng với giao diện Bootstrap, sử dụng các thành phần như menu, bảng giá, và giỏ hàng.
  2. Sử dụng grid system và các lớp Bootstrap để đảm bảo giao diện đáp ứng trên mọi thiết bị.
  3. Lưu file HTML với tên Lab7_[HọTênSinhViên].html vào ổ D.

Bài 8: Tối ưu hóa và bảo mật giao diện web

  • Các nguyên tắc tối ưu hóa hiệu suất giao diện web.
  • Các biện pháp bảo mật cho giao diện web, như bảo vệ khỏi tấn công XSS và CSRF.
  • Cách kiểm tra và tối ưu mã nguồn trang web.

Bài lab 8: Tối ưu hóa và bảo mật giao diện web

Hướng dẫn chi tiết:

  1. Kiểm tra và tối ưu hóa một trang web để giảm thời gian tải và tăng trải nghiệm người dùng.
  2. Cài đặt các biện pháp bảo mật đơn giản như xác thực form và bảo vệ dữ liệu.
  3. Lưu file HTML và JavaScript với tên Lab8_[HọTênSinhViên].html và Lab8_[HọTênSinhViên].js.
