Môn Lập Trình Giao Diện Web

**Giảng viên:** Thạc sĩ Hoàng Quốc Tuấn

Mục tiêu khóa học

  • Hiểu được cách xây dựng giao diện web chuyên nghiệp, đẹp mắt và dễ sử dụng.
  • Làm quen với các công cụ và thư viện hỗ trợ phát triển giao diện web (HTML, CSS, JavaScript, Bootstrap, jQuery, v.v).
  • Tạo ra các website với khả năng tương tác tốt, tương thích với nhiều thiết bị khác nhau (Responsive Design).
  • Phát triển kỹ năng làm việc nhóm, xây dựng và tối ưu hóa giao diện web.

Điều kiện học

  • Sinh viên cần có kiến thức cơ bản về tin học và sử dụng máy tính.
  • Không yêu cầu kinh nghiệm lập trình trước khi học.
  • Sinh viên cần tham gia đầy đủ các bài học lý thuyết và thực hành.
  • Sinh viên nên có máy tính cá nhân hoặc thiết bị có thể cài đặt phần mềm lập trình và duyệt web.

Nội dung chi tiết khóa học

  • Tổng quan về lập trình web.
  • Các thành phần cơ bản của một trang web: HTML, CSS, JavaScript.
  • Quá trình phát triển giao diện web.
  • Các công cụ hỗ trợ phát triển giao diện web.

Bài Lab: Làm quen với HTML và cấu trúc của một trang web. Tạo một trang web giới thiệu bản thân.

  • Các thẻ cơ bản: <div>, <span>, <h1>, <p>, <a>.
  • Cách tạo bảng, form và danh sách trong HTML.

Bài Lab: Tạo trang thông tin lớp học sử dụng các thẻ bảng, danh sách và form.

  • Khái niệm về CSS và cách áp dụng vào HTML.
  • Sử dụng các thuộc tính CSS: font, màu sắc, căn chỉnh, padding, margin.
  • Các phương pháp lựa chọn phần tử CSS: ID, class, thẻ.

Bài Lab: Thiết kế giao diện với CSS cho một trang giới thiệu công ty.

  • Khái niệm về thiết kế web đáp ứng.
  • Sử dụng Media Queries trong CSS.
  • Tạo giao diện thích ứng với các loại màn hình khác nhau.

Bài Lab: Thiết kế giao diện bán hàng trực tuyến tương thích trên nhiều thiết bị.

  • Giới thiệu về JavaScript và cách tích hợp vào HTML.
  • Các câu lệnh cơ bản: biến, hàm, vòng lặp.
  • Tương tác với người dùng qua các sự kiện (click, hover, submit).

Bài Lab: Viết JavaScript kiểm tra dữ liệu form đăng ký (tên, email, mật khẩu).

  • Giới thiệu về jQuery và thao tác với DOM.
  • Các hiệu ứng cơ bản: ẩn/hiện, chuyển động, thay đổi thuộc tính.
  • Sự kiện trong jQuery và AJAX cơ bản.

Bài Lab: Thêm hiệu ứng ẩn/hiện nội dung, AJAX hiển thị dữ liệu từ JSON.

  • Giới thiệu về Bootstrap và các thành phần cơ bản.
  • Grid System, Buttons, Forms, Navigation Bars.
  • Cách sử dụng các lớp CSS và JavaScript của Bootstrap.

Bài Lab: Tạo giao diện trang chủ doanh nghiệp với Bootstrap.

  • Kỹ thuật tối ưu hóa giao diện web.
  • Kiểm thử và sửa lỗi giao diện web.
  • Công cụ hỗ trợ kiểm thử giao diện web (Google Lighthouse).

Bài Lab: Kiểm thử hiệu suất và tối ưu hóa trang web với Google Lighthouse.
